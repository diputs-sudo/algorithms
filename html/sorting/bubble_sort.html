Bubble Sort

A simple comparison based sorting algorithm

Bubble Sort repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order. While simple to understand, it is not efficient for large datasets.

Complexity & Characteristics

Best: O(n) Average: O(n²) Worst: O(n²) Memory: O(1) Stable In-Place Classic

How Bubble Sort Works

Bubble Sort compares each pair of adjacent elements and swaps them if they are in the wrong order. After each pass, the largest unsorted element "bubbles" to its correct position at the end.

Step-by-Step Process

  • Start at the beginning of the array.
  • Compare adjacent elements.
  • Swap if the left element is greater than the right.
  • Repeat passes until no swaps are needed.

When To Use It

Good for educational purposes and very small datasets.

When NOT To Use It

Avoid for large datasets due to O(n²) complexity.

Pseudocode

for i from 0 to n - 1:
    for j from 0 to n - i - 1:
        if array[j] > array[j + 1]:
            swap(array[j], array[j + 1])
